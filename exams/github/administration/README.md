# GitHub Administration Certification

## Exam Overview

The GitHub Administration certification validates your ability to manage and administer GitHub environments at the organization and enterprise level. This exam covers user identity management, repository administration, Actions management, security features, and enterprise governance. It is designed for GitHub administrators, IT managers, and DevOps leads responsible for managing GitHub at scale.

**Exam Details:**
- **Duration:** 120 minutes
- **Format:** Multiple choice (65 questions)
- **Passing Score:** 70%
- **Cost:** $99 USD
- **Delivery:** Online proctored
- **Validity:** 3 years
- **Prerequisites:** None (GitHub Foundations recommended)
- **Retake Policy:** 24-hour wait for first retake, 14 days for subsequent

## Exam Domains

### Domain 1: Support GitHub Enterprise (15%)
- GitHub Enterprise Cloud vs Enterprise Server
- Support ticket management and escalation
- GitHub Enterprise support resources
- Migration tools and strategies
- Licensing and plan management

### Domain 2: Manage User Identities and Access (20%)
- SAML SSO configuration and management
- SCIM provisioning for automated user management
- Team synchronization with identity providers
- Organization membership and roles
- Enterprise Managed Users (EMU)
- Two-factor authentication enforcement
- SSH and GPG key management

### Domain 3: Manage Repositories (15%)
- Repository creation policies
- Repository visibility (public, private, internal)
- Branch protection rules and rulesets
- CODEOWNERS configuration
- Repository templates and auto-init
- Archive and transfer repositories
- Custom repository roles

### Domain 4: Manage GitHub Actions (20%)
- Organization and enterprise Actions policies
- Self-hosted runner management and groups
- Allowed actions and reusable workflows
- Secrets and variables at org/enterprise level
- Required workflows
- Actions usage and billing

### Domain 5: Manage GitHub Advanced Security (15%)
- GHAS licensing and enablement
- Code scanning administration
- Secret scanning configuration
- Dependabot administration
- Security overview and dashboards
- Security policies for organizations

### Domain 6: Enterprise Administration (15%)
- Enterprise account management
- Audit log and compliance
- IP allow lists
- GitHub Connect
- Data residency
- Enterprise policies and governance

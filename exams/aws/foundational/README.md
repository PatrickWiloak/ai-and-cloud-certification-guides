# AWS Foundational Certifications

Foundational-level certifications validate broad understanding of AWS Cloud concepts. No prior cloud experience is required — these are ideal starting points for anyone beginning their cloud journey.

## Available Certifications

### Active

| Certification | Exam Code | Duration | Cost | Focus |
|---|---|---|---|---|
| [Cloud Practitioner](cloud-practitioner-clf-c02/) | CLF-C02 | 90 min | $100 | Core AWS cloud concepts, services, billing |

### AI/ML Foundational

| Certification | Exam Code | Duration | Cost | Focus |
|---|---|---|---|---|
| [AI Practitioner](../genai/) | AIF-C01 | 90 min | $150 | AI/ML concepts, generative AI, AWS AI services |

## Who Should Start Here?

- **Career changers** entering cloud technology
- **Business professionals** needing cloud literacy
- **Students** beginning cloud studies
- **Technical professionals** from non-cloud backgrounds
- **Sales and marketing** teams working with cloud products

## Recommended Path

1. **Cloud Practitioner (CLF-C02)** — Start here for general AWS knowledge
2. **AI Practitioner (AIF-C01)** — Add this if interested in AI/ML
3. Then move to **Associate-level** certifications based on your career goals

## Study Tips

- These exams test breadth, not depth — focus on understanding concepts
- Use the AWS Free Tier for hands-on exploration
- Expect 4-6 weeks of part-time study for each exam
- No coding or deep technical knowledge required

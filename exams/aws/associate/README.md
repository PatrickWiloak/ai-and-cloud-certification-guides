# AWS Associate Certifications

Associate-level certifications validate hands-on cloud skills and are the most popular AWS certifications. They require practical experience designing, deploying, and operating AWS solutions.

## Available Certifications

### Active

| Certification | Exam Code | Duration | Cost | Focus |
|---|---|---|---|---|
| [Solutions Architect](solutions-architect-saa-c03/) | SAA-C03 | 130 min | $150 | Designing distributed systems and architectures |
| [Developer](developer-dva-c02/) | DVA-C02 | 130 min | $150 | Developing and maintaining AWS applications |
| [CloudOps Engineer](cloudops-engineer-soa-c03/) | SOA-C03 | 180 min | $150 | Cloud operations, monitoring, automation |
| [Data Engineer](data-engineer-dea-c01/) | DEA-C01 | 170 min | $150 | Data pipelines, analytics, data governance |
| [Machine Learning Engineer](ml-engineer-mla-c01/) | MLA-C01 | 170 min | $150 | ML model development, deployment, operations |

### Retired

| Certification | Exam Code | Status |
|---|---|---|
| [SysOps Administrator](sysops-administrator-soa-c02/) | SOA-C02 | Retired Sept 2025 — replaced by CloudOps Engineer (SOA-C03) |

## Recommended Experience

- 1+ years of hands-on AWS experience
- Understanding of core AWS services (EC2, S3, VPC, IAM)
- AWS Cloud Practitioner or equivalent knowledge

## Which Associate Cert Should You Choose?

- **Solutions Architect (SAA-C03)** — Best all-around starting point; covers architecture and design
- **Developer (DVA-C02)** — Best for software developers building on AWS
- **CloudOps Engineer (SOA-C03)** — Best for operations/SRE roles; includes hands-on exam labs
- **Data Engineer (DEA-C01)** — Best for data professionals working with pipelines and analytics
- **ML Engineer (MLA-C01)** — Best for ML practitioners using SageMaker and AWS AI services

## Study Tips

- Expect 6-10 weeks of preparation per exam
- Hands-on experience is essential — use AWS Free Tier
- Solutions Architect is the most recommended first associate cert
- Many services overlap between exams, making subsequent certs easier

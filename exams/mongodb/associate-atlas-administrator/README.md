# MongoDB Associate Atlas Administrator

## Exam Overview

The MongoDB Associate Atlas Administrator exam validates the ability to manage and administer MongoDB Atlas deployments. This certification demonstrates proficiency in cluster management, security configuration, data management, performance optimization, monitoring, and backup/restore operations in MongoDB Atlas.

**Exam Details:**
- **Exam Code:** MongoDB Associate Atlas Administrator
- **Duration:** 75 minutes
- **Number of Questions:** 62 multiple-choice questions
- **Passing Score:** 65%
- **Cost:** $150 USD
- **Delivery:** Online proctored
- **Validity:** 3 years
- **Prerequisites:** None (hands-on Atlas administration experience recommended)

## Exam Domains

### Domain 1: Cluster Management (25%)
- Cluster tiers (M0 through M700)
- Multi-cloud and multi-region deployments
- Auto-scaling configuration
- Cluster upgrades and modifications
- Electable and analytics nodes

**Key Concepts:**
- Free tier (M0) vs shared (M2/M5) vs dedicated (M10+)
- Cross-region replication and global clusters
- Vertical and horizontal auto-scaling
- Cluster pausing and resumption

### Domain 2: Security and Access (20%)
- VPC peering and private endpoints
- Database user management
- LDAP integration
- IP access lists
- Network security

**Key Concepts:**
- AWS PrivateLink, Azure Private Link, GCP Private Service Connect
- SCRAM and x.509 authentication
- Atlas organizations, projects, and teams
- API key management

### Domain 3: Data Management (20%)
- Atlas Search configuration
- Performance Advisor recommendations
- Index management
- Database profiler
- Data Explorer

**Key Concepts:**
- Search index creation and management
- Real-time performance panel
- Slow query analysis
- Data Federation and Online Archive

### Domain 4: Performance (15%)
- Performance Advisor
- Index optimization
- Query analysis
- Cluster sizing

### Domain 5: Monitoring and Alerts (10%)
- Atlas monitoring metrics
- Custom alerts
- Real-time performance panel
- Integration with third-party tools

### Domain 6: Backup and Restore (10%)
- Continuous backup
- Cloud backup snapshots
- Point-in-time restore
- Backup compliance policies

# NVIDIA Certified Associate - AI Infrastructure Operations (NCA-AIIO)

## Exam Overview

The NVIDIA Certified Associate - AI Infrastructure Operations certification validates foundational knowledge of managing and operating GPU-accelerated infrastructure for AI workloads, including GPU monitoring, container management, basic Kubernetes GPU operations, and troubleshooting.

**Exam Code:** NCA-AIIO
**Exam Duration:** 60 minutes
**Number of Questions:** 50-60 questions
**Exam Format:** Multiple choice
**Cost:** $135 USD
**Validity:** 2 years
**Prerequisites:** Basic understanding of Linux administration and data center operations

## Exam Domains

### Domain 1: GPU Fundamentals and Monitoring (25%)
- NVIDIA GPU architecture basics
- nvidia-smi and GPU monitoring tools
- DCGM fundamentals
- GPU health indicators and metrics
- Common GPU error types

### Domain 2: Container and Runtime Management (25%)
- NVIDIA Container Toolkit
- Docker with GPU support
- NGC container registry
- Container image management
- GPU resource allocation in containers

### Domain 3: Kubernetes GPU Operations (20%)
- NVIDIA GPU Operator basics
- GPU scheduling in Kubernetes
- Device plugin and feature discovery
- Basic MIG operations
- Pod GPU resource requests

### Domain 4: Infrastructure Management (15%)
- GPU driver management
- CUDA toolkit basics
- System requirements for GPU workloads
- Storage and networking for AI
- Environment configuration

### Domain 5: Troubleshooting and Maintenance (15%)
- Common GPU issues and resolution
- Log analysis for GPU errors
- Performance monitoring basics
- Maintenance procedures
- Escalation and documentation

# GitHub Administration Study Plan

## 4-Week Study Schedule

### Week 1: Enterprise and Identity
- [ ] Study GHEC vs GHES differences and use cases
- [ ] Learn SAML SSO configuration and IdP integration
- [ ] Study SCIM provisioning and team synchronization
- [ ] Understand Enterprise Managed Users (EMU)
- [ ] Learn organization and enterprise roles and permissions
- [ ] Review Notes: `notes/01-enterprise-identity.md`, `notes/02-repository-management.md`

### Week 2: Repositories and Actions
- [ ] Study repository visibility, creation policies, and templates
- [ ] Learn branch protection rules vs rulesets
- [ ] Study CODEOWNERS and custom repository roles
- [ ] Learn Actions policies and allowed actions configuration
- [ ] Study self-hosted runner management and security
- [ ] Review Notes: `notes/03-actions-management.md`

### Week 3: Security and Governance
- [ ] Study GHAS licensing model and enablement
- [ ] Learn code scanning and secret scanning administration
- [ ] Study Dependabot configuration and administration
- [ ] Learn audit log features, API access, and streaming
- [ ] Study enterprise policies and IP allow lists
- [ ] Review Notes: `notes/04-ghas-administration.md`, `notes/05-enterprise-governance.md`

### Week 4: Review and Exam Prep
- [ ] Deep review of identity management and Actions (40% combined)
- [ ] Practice exams and targeted review of weak areas
- [ ] Review all notes and flashcards
- [ ] Pre-exam preparation

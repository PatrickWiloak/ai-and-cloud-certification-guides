# Palo Alto Networks

## About Palo Alto Networks

Palo Alto Networks is a leading cybersecurity vendor founded in 2005 by Nir Zuk. The company pioneered the next-generation firewall (NGFW) and built its platform on a single-pass parallel processing (SP3) architecture that classifies traffic by application, user, and content rather than by port and protocol alone. Today the company sells across three platforms - Strata (network security and NGFW), Prisma (cloud security and SASE), and Cortex (security operations, XDR, XSOAR, XSIAM) - and is one of the largest pure-play cybersecurity companies by revenue.

Their certification program validates skills across these platforms. Most certifications are role-based and require the EDU-XXX official course or hands-on equivalent experience. Exams are delivered through Pearson VUE.

## Certification Family

### Network Security (Strata) Track

| Cert | Full Name | Audience | Notes |
|---|---|---|---|
| **PCCSA** | Palo Alto Networks Certified Cybersecurity Associate | Entry-level, broad cybersecurity awareness | Retired in many markets; replaced by PCCET |
| **PCCET** | Palo Alto Networks Certified Cybersecurity Entry-level Technician | Entry-level vendor-neutral cybersecurity | Successor to PCCSA |
| **PCNSA** | Palo Alto Networks Certified Network Security Administrator | NGFW administrator, day-to-day operations | EDU-210 Firewall Essentials |
| **PCNSE** | Palo Alto Networks Certified Network Security Engineer | NGFW engineer, deep configuration and troubleshooting | EDU-220 + EDU-330 |
| **PCSAE** | Palo Alto Networks Certified Security Automation Engineer | Cortex XSOAR playbook developer | EDU-380 |

### Cloud Security (Prisma) Track

| Cert | Focus |
|---|---|
| **Prisma Certified Cloud Security Engineer (PCCSE)** | Prisma Cloud (CSPM, CWP, IAM, Code Security) |
| **Prisma Access SASE Engineer / Architect** | Prisma Access, Prisma SD-WAN, SASE design |

### Security Operations (Cortex) Track

| Cert | Focus |
|---|---|
| **Cortex XDR Expert / Engineer** | Cortex XDR endpoint detection and response |
| **Cortex XSIAM Engineer** | Next-gen SIEM/SOC platform replacing legacy SIEMs |
| **PCSAE** | Cortex XSOAR automation, playbook authoring |

## Common Path

For someone entering Palo Alto Networks-aligned network security:

```
PCCET (optional foundation)
   ↓
PCNSA (administrator level, this guide)
   ↓
PCNSE (engineer level, design and deep troubleshooting)
   ↓
Specialize: Prisma Cloud, Prisma Access SASE, or Cortex XSOAR/XSIAM
```

For SOC analysts and automation engineers, the Cortex track (XDR → XSIAM → PCSAE) is more relevant than NGFW certs.

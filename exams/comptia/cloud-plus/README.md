# CompTIA Cloud+ (CV0-004)

## Exam Overview

The CompTIA Cloud+ (CV0-004) certification validates the skills needed to deploy, secure, optimize, and troubleshoot cloud infrastructure services. This vendor-neutral certification covers multi-cloud environments and is ideal for cloud engineers, system administrators, and network engineers working with cloud technologies.

**Exam Details:**
- **Exam Code:** CV0-004
- **Duration:** 90 minutes
- **Number of Questions:** Up to 90 questions
- **Question Types:** Multiple choice and performance-based questions
- **Passing Score:** 750 out of 900
- **Cost:** $369 USD
- **Language:** English
- **Delivery:** Pearson VUE testing center or online proctoring
- **Validity:** 3 years (renewable via CE program)
- **Prerequisites:** None required (2-3 years cloud experience recommended)

## Exam Domains

### Domain 1: Cloud Architecture (13%)
- Compare and contrast cloud deployment models
- Determine appropriate cloud service models
- Implement cloud design requirements for high availability and disaster recovery
- Analyze cloud design for security, performance, and cost optimization

**Key Topics:**
- Deployment models - public, private, hybrid, multi-cloud, community
- Service models - IaaS, PaaS, SaaS, XaaS, FaaS, DBaaS
- High availability concepts - redundancy, fault tolerance, replication
- Design principles - scalability, elasticity, loose coupling

### Domain 2: Security (20%)
- Implement security controls and compliance
- Manage identity and access in cloud environments
- Secure data in cloud environments
- Implement network security in cloud environments

**Key Topics:**
- Identity and access management - RBAC, ABAC, federation, SSO, MFA
- Data security - encryption at rest, encryption in transit, key management
- Network security - firewalls, security groups, NACLs, VPNs, micro-segmentation
- Compliance frameworks - SOC 2, ISO 27001, HIPAA, PCI-DSS, GDPR, FedRAMP

### Domain 3: Deployment (23%)
- Integrate components into cloud solutions
- Provision cloud storage and compute resources
- Apply automation and orchestration techniques
- Execute migration strategies

**Key Topics:**
- Migration types - lift-and-shift, replatform, refactor, retire, retain
- Automation tools - Terraform, Ansible, Puppet, Chef
- Infrastructure as Code - templates, version control, state management
- Containers and orchestration - Docker, Kubernetes, container registries
- CI/CD pipelines and deployment strategies

### Domain 4: Operations and Support (22%)
- Operate and maintain cloud environments
- Optimize cloud environments
- Apply troubleshooting methodologies
- Manage cloud resources and capacity

**Key Topics:**
- Monitoring and logging - metrics, alerts, dashboards, log aggregation
- Optimization - right-sizing, reserved instances, cost management
- Change management - ITIL processes, configuration management
- SLA management - uptime guarantees, RTO/RPO, performance metrics
- Patching and updates - rolling updates, maintenance windows

### Domain 5: Troubleshooting (22%)
- Troubleshoot connectivity issues
- Troubleshoot security issues
- Troubleshoot deployment issues
- Troubleshoot performance issues

**Key Topics:**
- Network connectivity - DNS, routing, firewall rules, VPN tunnels
- Performance issues - CPU, memory, storage I/O, network throughput
- Security incidents - unauthorized access, misconfigurations, data exposure
- Automation failures - script errors, template issues, state drift
- Systematic troubleshooting methodology

# Salesforce Certified Administrator

## Overview

The Salesforce Certified Administrator credential validates the ability to configure, manage, and customize a Salesforce org. It's the foundational Salesforce cert and a prerequisite for many advanced credentials (Advanced Administrator, Sales Cloud / Service Cloud Consultant, etc.).

This is the most-requested Salesforce cert on job postings and the typical entry point into the Salesforce ecosystem.

---

## Quick Reference

| Detail | Info |
|---|---|
| **Exam Name** | Salesforce Certified Administrator |
| **Provider** | Salesforce |
| **Format** | 60 multi-choice + multi-select questions |
| **Duration** | 105 minutes |
| **Passing Score** | 65% |
| **Cost** | $200 USD (+ $100 retake) |
| **Validity** | Maintenance via 3 free annual release exams (Spring/Summer/Winter), no expiration if maintained |
| **Delivery** | Onsite or online proctored |

## Hands-on Setup

- **[Salesforce Developer Edition](https://developer.salesforce.com/signup)** - free permanent dev org with most features
- **[Trailhead Playgrounds](https://trailhead.salesforce.com/)** - launch scratch orgs from any module
- Recommended Trailmix: **["Prepare for Your Salesforce Administrator Credential"](https://trailhead.salesforce.com/users/strailhead/trailmixes/administrator-certification-prep)**

You **cannot** pass this cert without hands-on time. Most exam questions are scenario-based and test your judgment on real Salesforce setup.

## After Administrator

Common next steps:

- **Advanced Administrator** - deeper admin (formula fields, complex sharing, advanced automation)
- **Salesforce Platform App Builder** - declarative app development
- **Sales Cloud Consultant** - sales-focused implementation
- **Service Cloud Consultant** - service-focused implementation
- **Salesforce Platform Developer I (PD1)** - cross over to development with Apex, LWC
- **CRM Analytics Consultant** (formerly Tableau CRM)

# Microsoft Certified: Microsoft 365 Fundamentals (MS-900)

## Overview

The Microsoft 365 Fundamentals certification validates foundational knowledge of Microsoft 365 services, productivity solutions, security, compliance, privacy, and trust principles. It's a non-technical entry-point for sales, support, business decision-makers, and IT professionals beginning their M365 journey.

---

## Quick Reference

| Detail | Info |
|---|---|
| **Exam Code** | MS-900 |
| **Full Name** | Microsoft Certified: Microsoft 365 Fundamentals |
| **Provider** | Microsoft |
| **Format** | Multiple choice, multiple response (no labs) |
| **Duration** | 65 minutes (~45 questions) |
| **Cost** | $99 USD |
| **Validity** | Does not expire (perpetual) |
| **Passing Score** | 700 / 1000 |
| **Delivery** | Online proctored or testing center |

## Recommended Study Time

| Background | Estimated Prep Time |
|---|---|
| Existing M365 admin or user | 1-2 weeks |
| IT background, no M365 experience | 3-4 weeks |
| New to IT | 5-6 weeks |

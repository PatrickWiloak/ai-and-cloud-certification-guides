# AWS Shared Service Documentation

This directory contains reusable AWS service documentation referenced across multiple certification study guides. Instead of duplicating service details in every exam directory, shared files provide a single source of truth.

## How to Use

These files are referenced from individual certification study guides. You can also use them as standalone quick-reference documentation for AWS services.

## Service Index

### Compute
- EC2, Lambda, ECS/EKS, Elastic Beanstalk, Auto Scaling

### Storage
- S3, EBS, EFS, FSx, Storage Gateway

### Database
- RDS, Aurora, DynamoDB, ElastiCache, Redshift

### Networking
- VPC, Route 53, CloudFront, Direct Connect, Transit Gateway, ELB

### Security & Identity
- IAM, KMS, Secrets Manager, WAF, Shield, GuardDuty, Security Hub

### Management & Governance
- CloudWatch, CloudTrail, Config, Systems Manager, Organizations, Control Tower

### Application Integration
- SQS, SNS, EventBridge, Step Functions, API Gateway

### Analytics & ML
- Athena, Glue, Kinesis, EMR, SageMaker, Bedrock
